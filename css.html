CSS - Cascading Style Sheets

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência de um documento HTML. Com CSS, você pode controlar as cores, fontes, layouts e até a posição dos elementos na página.

A principal função do CSS é separar a estrutura do conteúdo (HTML) da apresentação visual. Isso facilita a manutenção do código e a criação de websites responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

CSS3, a versão mais recente do CSS, introduziu novas funcionalidades como animações, transições, sombras e muito mais.

Principais Propriedades do CSS

  • color - Define a cor do texto.
  • background-color - Define a cor de fundo de um elemento.
  • font-size - Define o tamanho da fonte.
  • margin - Define a margem externa de um elemento.
  • padding - Define o preenchimento interno de um elemento.
  • display - Controla o modelo de layout de um elemento (por exemplo, block ou inline).
