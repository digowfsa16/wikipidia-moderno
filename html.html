HTML - Linguagem de Marcação

O HTML (HyperText Markup Language) é a base da construção de páginas web. Ele é utilizado para estruturar conteúdo e fornecer a semântica necessária para os navegadores interpretarem e renderizarem as páginas corretamente.

HTML não é considerado uma linguagem de programação, pois não possui lógica de controle. Ele é uma linguagem de marcação utilizada para criar elementos como parágrafos, links, imagens, tabelas, e outros componentes estruturais de uma página web.

HTML5 é a versão mais recente do HTML, que inclui novos elementos, atributos e APIs, proporcionando maior flexibilidade e controle na criação de websites modernos, incluindo suporte nativo para multimídia (áudio e vídeo) e gráficos.

Principais Elementos do HTML

  • <h1>, <h2>, <h3>, etc. - Títulos e subtítulos.
  • <p> - Parágrafos de texto.
  • <a> - Links para outras páginas ou recursos.
  • <img> - Imagens.
  • <div> - Elemento de bloco utilizado para agrupar conteúdo.
